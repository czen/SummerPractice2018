Программа поиска изоморфизма подграфа

1. Запустите программу через командную строку из папки Release, расположенной в корне проекта, указав вторым параметром один из текстовых файлов, кроме output.txt.
Если программа запущена не через командную строку, то по умолчанию запустится файл input.txt.
2. Все текстовые файлы содержат 2 матрицы смежности графов и имеют строго следующую структуру:
	а) сначала указана размерность матрицы первого графа (ЭТОТ ГРАФ ДОЛЖЕН БЫТЬ БОЛЬШИМ ИЗ ДВУХ ВВОДИМЫХ ПОЛЬЗОВАТЕЛЕМ);
	б) затем следует его матрица смежности, которая не содержит никаких пробелов или знаков между элементами;
	в) следом (без пустых строк между предыдущей матрицей) указывается размерность подграфа;
	г) и, аналогично пункту б), следует матрица смежности подграфа.
3. Все выходные данные программы сгенерируются в файле output.txt.Там появятся три результата решения задачи.

Далее перечислены значения изоморфности текстовых файлов, которые должны быть на выходе:
	input.txt - не изоморфны
	test1.txt - изоморфны
	test2.txt - изоморфны (можно проследить на рисунке, что у каждой вершины только 3 ребра)
	test3.txt - не изоморфны (изменен первый граф из test2.txt, у него из 17 вершины 4 ребра)
	test4.txt - не изоморфны
В папке Схемы графов в корне проекта приложены изображения графов из тестов для удобства визуальной проверки.
