К программе прилагаются 3 теста - tree.txt, tree_example.txt, test.txt. 
Для проверки работы алгоритма необходимо присоединить один из них к проекту через его свойства (аргументы командной строки).

При запуске проекта на консоли можно выбрать вариант реализации алгоритма - через рекурсию (R) или с использованием стека (S).
Помимо вывода на экран результат работы автоматически запишется в файл result.txt. 

Перед запуском через командную строку убедитесь, что файл ввода и файл вывода находятся в одном каталоге.
